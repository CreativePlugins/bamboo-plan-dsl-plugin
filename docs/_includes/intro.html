Plan DSL for Bamboo API reference

Welcome to the Plan DSL for Bamboo API reference! In the following sections you can find the structure and building elements of the Groovy-based DSL and YAML syntax which allows you to manage your Bamboo build plans and deployment projects as code.

The project is hosted on GitHub. You can find more information about the setup of the plug-in, real-world use cases and advanced concepts in the Wiki.
